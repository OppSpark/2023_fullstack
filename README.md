# 첫 풀스택 프로젝트
- 2023년 1학기 여름방학 스터디 후 배운 내용을 토대로 만든 간단한 게시판 입니다.
- 첫 프로젝트인 만큼 부족한 것이 많습니다.


## 프로젝트 소개
- 프론트엔드는 HTML, CSS, JS 바닐라 3종을 사용했습니다.
- 백엔드는 node.js, Express를 사용했습니다.
- DBMS는 MySQL을 사용했습니다.
- 웹서버는 Nginx를 사용했습니다.
- 서버는 Amazone EC2 를 사용했습니다.

## 구현 기술 소게
- 회원가입, 로그인 기능을 구현했습니다.
- 회원가입, 로그인 시 예외처리를 했습니다.
- 게시판 작성 기능을 구현했습니다.
-  반복문을 이용해 게시글을 출력했습니다.
